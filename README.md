# Simple To-Do app
- PostgreSQL, Express, React, Node
- TypeScript

## Features
- Register / Login
- Add new tasks
- Edit existing tasks (double click) (click outside to save)
- Delete tasks
- Browse all, active, completed tasks
- Automatic Light / Dark mode
